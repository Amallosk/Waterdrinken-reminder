# Changelog

All notable changes to Kos's Hydration Pit Stop will be documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-03-26

### Added

- Racing-themed water drinking reminder app (single HTML file)
- Speedometer gauge showing progress toward 8 glasses/day
- "IK HEB WATER GEDRONKEN!" button with confetti explosion and race car animation
- 60-minute alarm with fullscreen overlay and siren sound (Web Audio API)
- Dashboard with last pit stop, glasses today, and countdown timer
- Pit stop log with timestamps
- Extra confetti celebration when daily goal is reached
- localStorage persistence (survives page reload)
- Automatic day reset at midnight
- Browser notification support (optional)
- CLAUDE.md with project conventions and roadmap
